# Cullery Full Spec

## 1. 文档目标

本文件用于描述 Cullery / 览羲 的完整产品蓝图，覆盖移动端、桌面端、专业选片、语音控制、模型管理与长期演进能力。

## 2. 产品愿景

Cullery 不仅是照片清理工具，更是一个离线 AI 照片决策系统。它既要帮助用户释放存储空间，也要替用户做更专业、更可信的保留决策。

## 3. 完整产品范围

### 3.1 基础管理能力

- 空间分析
- 截图清理
- 重复项识别
- 相似组清理
- 低质量照片识别
- 应用内回收站

### 3.2 专业选片能力

- 多维摄影质量评分
- 人像 / 风景 / 活动 / 旅行场景自适应
- 相似组最佳单张推荐
- 组照叙事推荐
- 个性化偏好学习
- 推荐原因解释

### 3.3 语义理解能力

- 基础语音控制
- 自然语言命令解析
- 按时间、地点、场景、人物进行检索与筛选
- 支持“删除去年所有模糊照片”这类复合命令

### 3.4 模型与智能能力

- 轻量模型本地运行
- 1B/2B 多模态模型增强
- 7B 量化模型异步能力
- 模型下载、升级、缓存、状态管理

### 3.5 平台能力

- Android
- iOS
- Desktop
- Web/PWA 演示版

### 3.6 高级能力

- 精华故事集
- 时间线视图
- 地图视图
- 事件相册自动创建
- 桌面端批量管理
- 导出到专业工具

## 4. 专业选片全量规格

### 4.1 技术维度

- 清晰度
- 失焦情况
- 噪点水平
- 曝光平衡
- 高光与阴影细节
- 色彩准确性
- 白平衡
- 动态范围
- 镜头畸变与色差

### 4.2 视觉设计维度

- 三分法
- 视觉平衡
- 引导线
- 负空间
- 对称性
- 主体完整度
- 画面干扰度

### 4.3 情感与叙事维度

- 微表情质量
- 肢体互动
- 关键瞬间
- 情绪浓度
- 环境氛围
- 故事完整性

### 4.4 场景与时空维度

- 白天 / 黑夜
- 室内 / 室外
- 季节
- 天气
- 地点层级
- 地标价值
- 背景杂乱度
- 旅行 / 家庭 / 聚会 / 会议 / 餐饮等场景分类

## 5. 评分与推荐逻辑

### 5.1 推荐流程

- 第一层：废片淘汰
- 第二层：技术评分
- 第三层：构图与场景评分
- 第四层：情感与叙事评分
- 第五层：场景加权推荐

### 5.2 推荐输出

- 推荐保留 1 张
- 备选保留 1-2 张
- 推荐删除若干张
- 每张图片的评分雷达图
- 可解释推荐理由

## 6. 平台演进路线

### 6.1 Android

- 首发与主验证平台
- 优先打通真实相册、删除、模型下载与语音

### 6.2 iOS

- 后续适配 PhotoKit、Speech、文件系统差异
- 强化隐私与本地处理叙事

### 6.3 Desktop

- 大屏批量管理
- 多图对比
- 导入导出
- 与专业工具协作

## 7. 商业化规格

### 7.1 国内方向

- 基础功能免费
- 高级整理与模型能力订阅
- 中文语音与证件类能力增强

### 7.2 国际方向

- 试用 + 订阅 / 买断
- 强调 `No Cloud, No Tracking`
- 更重视家庭回忆与旅行内容策展

## 8. 完整风险

- 7B 模型移动端落地困难
- 下载与存储空间压力大
- 专业选片存在主观性分歧
- 不同设备系统权限差异明显
- 自动删除必须极度谨慎

## 9. 完整验收标准

- 支持移动端完整清理闭环
- 支持至少一类高级语音命令
- 支持至少一类场景化推荐
- 支持模型真实下载与本地状态管理
- 支持桌面端扩展接口预留
- 支持中英文完整文案一致性

## 10. Full Spec 完成定义

当产品从“清理工具”升级为“离线 AI 记忆策展人”，并具备：

- 基础清理能力
- 专业选片能力
- 场景理解能力
- 语音控制能力
- 模型管理能力
- 跨平台演进能力

即可认为进入 Full Spec 实施阶段。
